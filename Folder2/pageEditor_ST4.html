Challenges and Lessons Learned

Major challenges involved in designing the web page content included implementing the sitemap using SVG elements, while ensuring proper alignment, structure, and interactivity within the layout. Additionally, maintaining consistency with the overall design by following styling properties defined in the external CSS file 'Global.css' required careful adjustments. However, this proved to be a valuable part of the project since it encouraged further research into SVG usage and allowed experimentation with different styling techniques while testing other code components independently.
