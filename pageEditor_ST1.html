Home page

Technical Description

For the implementation of the Home page, key html tags used involved <header>, within which the main navigation <nav> links were placed. Unordered lists were used for the navigation menu, implemented with <ul> and <li> tags. <div> tags were used for grouping minor collections of content, including images and text with <p> tags. Within the <main> tag, several <section> tags were embedded to seperately showcase the home page card content, as well as the content thumbnails in the gallery. Heading tags of different types were used, including <h2> and <h3>. <a> tags were used for inserting hyperlinks to other content pages. The <footer> tag was finally implemented within the <body> tag to denote the page logo, alongside links to all other pages. CSS styling was used internally for the gallery thumbnails, for which both classes and id's were used, while styling for the navigation and card content were done through an external CSS sheet 'Global.css'

Accessibility

Use of semantic HTML tags, such as <section> and a clear, readable color palette was used to for accessibility. Furthermore, 'alt' attributes were used to provide descriptions for all images showcased in the page.
